# 実装内容

## 必須課題

JenkinsからPythonスクリプトを実行し、天気データをJSON・CSVへ出力する。GitHubのpushイベントはngrok経由のWebhookでJenkinsへ通知し、自動ビルドを開始する。

## 加点要素

- 授業で扱っていないJavaScriptを用いてHTMLダッシュボードを生成
- Pythonの単体テストを実行
- JSON・CSV・HTMLの整合性を自動検証
- 成功・失敗時のDiscord通知（Webhook設定時のみ）
- Jenkins成果物として4ファイルをアーカイブ
- API障害時のサンプルデータ切替

## 処理の流れ

1. GitHubへpush
2. GitHub WebhookからngrokへPOST
3. ngrokがJenkinsの `/github-webhook/` へ転送
4. Jenkinsがリポジトリをcheckout
5. Python単体テスト
6. Open-Meteoから天気取得、JSON・CSV出力
7. Node.jsでHTML生成
8. Pythonで成果物検証
9. Jenkinsが成果物をarchive
10. 設定済みの場合はDiscordへ結果通知
