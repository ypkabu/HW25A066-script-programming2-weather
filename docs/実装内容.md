# 実装内容

## 必須課題

JenkinsからPythonスクリプトを実行し、天気データをJSON・CSVへ出力する。GitHubのpushイベントはngrok経由のWebhookでJenkinsへ通知し、自動ビルドを開始する。

## 加点要素

- 授業で扱っていないJavaScriptを用いてHTMLダッシュボードを生成
- Pythonの単体テストを実行
- JSON・CSV・HTMLの整合性を自動検証
- 成功・失敗時のDiscord通知（Webhook設定時のみ）
- 高温・低温・強い雨・強風の気象警告JSONとHTML表示
- 前回ビルドとの差分JSON（平均気温差・降水量差・変化日数）
- 成果物マニフェストJSON（ファイル名・サイズ・SHA-256・生成日時・ビルド番号）
- Jenkins成果物として全outputファイルをアーカイブ
- API障害時のサンプルデータ切替

## 処理の流れ

1. GitHubへpush
2. GitHub WebhookからngrokへPOST
3. ngrokがJenkinsの `/github-webhook/` へ転送
4. Jenkinsがリポジトリをcheckout
5. Python単体テスト
6. Open-Meteoから天気取得、JSON・CSV出力
7. Node.jsでHTML生成
8. Pythonで成果物検証
9. 気象警告・前回差分・成果物マニフェストを生成
10. Jenkinsが成果物をarchive
11. 設定済みの場合はJenkins CredentialsのSecret textからDiscordへ結果通知
