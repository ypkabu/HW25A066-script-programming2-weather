# Jenkins実行動画の撮影手順

授業資料では、GitHub・ngrok・Jenkinsの3アプリケーションの連動が分かる映像が必要です。次の順番で1本の動画にします。

1. Jenkinsのジョブ画面を表示する。最新ビルド番号を見せる。
2. ngrokのPowerShell画面を表示し、Forwarding URLと接続中であることを見せる。
3. GitHub Desktopまたはターミナルで、READMEに確認用の1行を追加する。
4. commitしてpushする。
5. ngrok画面で `POST /github-webhook/ 200 OK` が出るところを見せる。
6. Jenkinsへ戻り、新しいビルドが自動開始したことを見せる。
7. Console Outputを開き、次を順に見せる。
   - Unit tests: OK
   - Weather data output
   - JavaScript dashboard extension
   - Verify outputs: success
   - Finished: SUCCESS
8. Artifactsを開き、`weather_data.json`、`weather_data.csv`、`index.html`、`build_summary.json` を見せる。
9. `index.html` をブラウザで開いて、7日分のカードと表を見せる。

Windowsでは `Win + Alt + R` でXbox Game Barの録画を開始・終了できます。

注意: ngrokのauthtoken、GitHubのアクセストークン、Discord Webhook URLは動画に映さないこと。
