このフォルダには、課題の実装・出力・設定手順・提出用レポートをまとめています。

すでに完成しているもの:
・天気データ出力プログラム
・JavaScriptによるHTMLダッシュボード
・Jenkinsfile
・単体テスト5件
・成果物検証
・Discord通知（任意設定）
・JSON / CSV / HTML / build_summary のサンプル出力
・ローカル実行ログ
・提出用レポート

本人PCで最後に必要なもの:
・GitHub push → ngrok → Jenkins自動起動を映した実行動画
・その成功ビルドからダウンロードしたJenkinsコンソールログ

この2つは、GitHub・ngrokアカウントと本人PC上のJenkinsが必要なため、外部では正規の証跡を生成できません。
動画とログを取得後、make_submission.ps1を実行すると提出ZIPが自動作成されます。
